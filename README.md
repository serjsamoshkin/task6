# task 6 (block 08)
## Задание 1.
  Есть ArrayList&lt;Integer&gt; - ов. За один проход подсчитать количество повторений каждого элемента.
  Т.е., если мы имеем массив с элементами:
  4,5,-6,4,5,3,4,2,4,5,7
  То получим структуру:
  -6 – 1
  2 – 1
  3 – 1
  4 – 4
  5 – 3
  7 – 1

  нельзя удалять.
